RestaurantFlow — высоконагруженное микросервисное приложение для автоматизации ресторанного бизнеса (от интерактивного меню до процессинга платежей). Разработано с использованием событийно-ориентированной архитектуры (EDA) и чистой многослойной структуры (DDD).  

Стек технологий: Python 3.10+, FastAPI, SQLAlchemy (Async) , PostgreSQL (Multi-schema) , RabbitMQ (aio-pika) , Redis , Docker / Docker Compose (Multistage builds) , Alembic , Traefik API Gateway , Pydantic v2 , MinIO S3.
